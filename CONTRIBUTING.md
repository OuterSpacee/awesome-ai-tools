# Contributing to Awesome AI Tools

Thank you for helping make this the best AI tools directory on the internet.

## How to Add a Tool

1. **Fork** this repository
2. **Add** your tool to the correct category in `README.md`
3. Follow the **existing table format** exactly
4. **Submit** a pull request

## Criteria for Inclusion

A tool should meet these criteria:

- **Actively maintained** - last update within 6 months
- **Publicly accessible** - has a website or public repository
- **Relevant** - directly related to AI/ML for developers or users
- **Not a duplicate** - doesn't replicate a tool already listed
- **Functional** - actually works and provides value

## Formatting

### For tools with a website:
```markdown
| [Tool Name](https://tool-url.com) | Brief description (max 80 chars) | 💰 🌐 |
```

### For open-source tools:
```markdown
| [Tool Name](https://github.com/org/repo) | Brief description (max 80 chars) | 🟢 🔓 |
```

### Icons:
| Icon | When to Use |
|------|-------------|
| 🟢 | Completely free, no paid tier |
| 💰 | Has a free tier, but also paid plans |
| 💵 | Paid only, no free tier |
| 🔓 | Open source (has public source code) |
| 🌐 | Web-based / browser |
| 🖥️ | Desktop application |
| ⌨️ | CLI / Terminal tool |

## Quality Standards

- Descriptions should be concise and factual
- Don't use superlatives ("best", "amazing", "revolutionary")
- Don't include referral links or affiliate URLs
- Tools should be listed alphabetically within each category unless there's a clear market leader ordering

## New Categories

If a tool doesn't fit any existing category, propose a new one in your PR description. Categories should have at least 5 tools to be added.

## Reporting Issues

- **Broken link?** Open an issue with the tool name and broken URL
- **Tool discontinued?** Open an issue or PR to remove it
- **Wrong category?** Open a PR to move it

## Code of Conduct

Be respectful. No self-promotion spam. Focus on quality over quantity.
